# **App Name**: ZenithFlow

## Core Features:

- Mode Selection: Allow users to select between Study, Focus, and Relax modes to tailor their experience.
- Mood Assessment: Prompt users to rate their current mood to personalize tips, music, and motivational messages.
- Focus Timer: Offer timed sessions (15 min, 30 min, 1 hour) with integrated music to enhance focus, tailored to the selected mode.
- AI-Powered Personalized Tips: Leverage generative AI tool to provide mode-specific focus tips, pulling from a knowledge base based on the mode and userâ€™s self-reported mood. Tips can incorporate if/when to include or ignore specific information.
- Motivational Messaging: Deliver simple motivational messages before and during sessions to encourage users and maintain engagement.
- Monthly Progress Wrap-Up: Compile a monthly summary of user activity and progress, showcasing session statistics and overall growth. Include time in different modes.
- Reward System: Implement a reward system that gifts users a virtual reward upon successful completion of a focus mode, encouraging continued use.

## Style Guidelines:

- Primary color: Soft lavender (#E6E6FA) for a calming and fresh feel.
- Background color: Pale grey (#F5F5F5) for a clean, minimalistic background that doesn't distract from the content.
- Accent color: Light sky blue (#87CEEB) to highlight interactive elements and provide visual cues.
- Body and headline font: 'Inter' sans-serif for a modern, neutral and readable design.
- Use simple, line-based icons that reflect the theme of each mode (e.g., a book for Study, a target for Focus, sound waves for Relax).
- Employ a clean, card-based layout to organize content, ensuring ease of use and visual appeal.
- Subtle animations for transitions and feedback, enhancing user engagement without being distracting.
